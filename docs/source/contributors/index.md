# jupyter-ai-persona-manager

The **persona manager** is the core registry and lifecycle manager for AI
personas in Jupyter AI. A _persona_ is an AI participant in a chat — with its own
name, avatar, model, and behavior. Multiple personas can coexist in the same
chat and even mention one another.

New to personas? Start with the [Overview](./overview.md), then follow
[Developing a custom persona](./custom-personas.md). The
[Architecture](./architecture.md) page explains how it all fits together, and
the [Python API](./python-api.md) and [REST API](./rest-api.md) are the
generated references.
